Racetracksim
============


Preparation
-----------

Befoer building you'll need to install https://cmake.org/.

Dependencies
------------

Free glut

Build it yourself:
http://freeglut.sourceforge.net/index.php#download

Or pre-packaged:
http://www.transmissionzero.co.uk/software/freeglut-devel/


Build
-----

```bash
# Clone this repository.
mkdir build
cd build
cmake ..
make
```
